# android studio 在mac 上的快捷键
- 查看类继承关系 control + h
- 查看当前类有哪些方法 command + 7

# 其他
- 显示隐藏文件 shift +command + 。
- 截屏 shift +command + 3/4/5/6
- 关闭当前窗口 command + Q
- 文件排序 control +command +1/2/3
- 打开某个文件（Terminal 输入） open + 文件名
- 安装git（Terminal 输入）git
